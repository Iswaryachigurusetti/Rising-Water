## 7.1 Model Performance Comparison

Trained and evaluated on the `flood_dataset.xlsx` dataset (115 rows,
92 train / 23 test, stratified 80/20 split):

| Model | Accuracy | Precision | Recall | F1 Score |
|---|---|---|---|---|
| Decision Tree | 95.65% | (see `models/metrics.json`) | (see `models/metrics.json`) | (see `models/metrics.json`) |
| Random Forest | 95.65% | (see `models/metrics.json`) | (see `models/metrics.json`) | (see `models/metrics.json`) |
| KNN | 86.96% | (see `models/metrics.json`) | (see `models/metrics.json`) | (see `models/metrics.json`) |
| XGBoost | Varies by run (typically 90%+) | (see `models/metrics.json`) | (see `models/metrics.json`) | (see `models/metrics.json`) |

> Full precision/recall/F1/confusion-matrix numbers for every run are written
> to `5. Project Development/models/metrics.json` each time
> `src/train_models.py` is executed, since results can shift slightly given
> how small (115-row) and imbalanced (~14% flood-positive) this dataset is.

**Best model selected:** Decision Tree (auto-selected by the pipeline based
on highest test accuracy, tie-broken by F1 score). The pipeline re-evaluates
and re-selects automatically any time the dataset or models change — this
isn't hardcoded.

## 7.2 Exploratory Data Analysis Outputs

Generated by `src/eda.py`, saved under
`5. Project Development/notebooks/plots/`:

1. **Class Distribution** — confirms the dataset is imbalanced, with
   flood-positive years being a minority class (~14%).
2. **Correlation Heatmap** — shows strong correlation between `ANNUAL`
   rainfall and its seasonal components (`Jun-Sep`, `Mar-May`, etc.), as
   expected since `ANNUAL` is their sum.
3. **Annual Rainfall Distribution** — right-skewed, with most years
   clustering in a moderate rainfall band and a smaller number of
   high-rainfall outlier years.
4. **Seasonal Rainfall Averages** — `Jun-Sep` (monsoon) contributes the
   overwhelming majority of annual rainfall, consistent with the Indian
   monsoon pattern.
5. **Temperature / Humidity / Cloud Cover Distributions** — roughly
   unimodal distributions, useful as secondary predictive signals alongside
   rainfall.
6. **Annual Rainfall vs. Flood Occurrence (Boxplot)** — flood years show a
   visibly higher median annual rainfall than non-flood years, validating
   rainfall as a strong predictive feature.
